1、程序 = 数据结构 + 算法

2、数据结构：由数据的逻辑结构、物理结构和运算组成。

3、算法的特性
    ●有穷性
    ●确定性
    ●可行性
    ●输入：一个算法有零个或多个输入
    ●输出：一个算法有一个或多个输出

4、好算法的特质
    ●正确性
    ●可读性
    ●健壮性：对非法数据能够做出适当处理
    ●高效率和低储存需求（即低的时间复杂度和空间复杂度）    

5、O(1) < O(㏒₂n) < O(n) < O(n㏒₂n) < O(n²) < O(n³) < O(2ⁿ) < O(n!) < O(nⁿ) （即 常对幂指阶 越往后越大）
